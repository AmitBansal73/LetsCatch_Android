Brief about Meet us Application:-

This Application is designed to catch a Person location by using their mobile location by sent a request from this application. GPS is intended to give the request received mobile location in terms of longitude and latitude on the Earth and get direction to their location. You can Track a single Contact or Multiple Contacts Geospatially at the same time, Organize Single or Group Meetings and Send your location to Configured Contacts In Case of emergency with programmed panic button, helps in schedule Of Group meetings.

Instructions to use Application:-

  1. Panic Alarm:- In case of any Emergency situation your location is sent to Configured contacts with a predefined message, by pressing this button.

    Go to Drawer --> settings --> Panic button

    Configure now

  2. Instant Meeting:- This feature help you to get a direction to selected Contact location and also track that contact , you can track multiple contacts at the same time.

    + Button --> Contact

    Track now

  3. Schedule Meeting:- This feature help you to Schedule a spatial Meetings with Date, Time and Location, The application will remind you at that time.

    Go to Drawer --> Schedule Meetings

    Schedule Now

  4. Schedule Repeat Meeting:- This feature will help you to set a Repeat Meeting you can repeat a meeting by Daily, Weekly, And Monthly.

    Go to Drawer --> Schedule Meetings (Click on Check button Repeat)

  5. Group Meetings :- Group option in Application will help you to Add a Group with the Selected Contacts, User Can track location of all the Contacts inside the Group at the same time by sending a request to them.

    +Button -->Group

    Add a Group Now
